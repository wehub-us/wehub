# WeHub

WeHub 是一个 AI 原生协作系统。

我们正在建设人类与智能体团队共同工作的底层基质：agent runtime、共享记忆、project seed、工作流协议，以及可公开沉淀的研究与工程 artifact。目标是让协作更有上下文、更可检查，也更能长期生长。

WeHub is an AI-native collaboration system.

We are building the working substrate for human-agent teams: agent runtimes, shared memory, project seeds, workflow protocols, and public research artifacts that make collaboration more contextual, inspectable, and durable.

## 公开 Seed / Public Seed

这个仓库是 WeHub 在 GitHub 上的第一个公开 seed repo。

它一开始就保持很小，是为了：

- 说明 WeHub 的公开身份；
- 把公司 / 系统级入口与 causal intelligence 研究线连接起来；
- 在 runtime、research、demo 等具体仓库成熟之前，提供一个稳定 landing point；
- 让未来结构可以自然 grow，而不是假装整个内部系统已经完整公开。

This repository is the first public seed for the WeHub organization on GitHub.

It starts small on purpose:

- clarify the public identity of WeHub;
- link the company/system-level org with the causal intelligence research line;
- provide a stable landing point before individual runtime, research, and demo repositories are ready;
- keep future structure growable instead of pretending the whole system is already public.

## 仓库分层 / Repository Layers

预期的公开分层：

- `runtime`：agent runtime、gateway 与协作基础设施；
- `research`：causal intelligence、eval 与研究 artifact；
- `labs`：原型、seed demo 与早期实验；
- `docs`：公开协议、说明文档与操作指南。

Expected public layers:

- `runtime`: agent runtime, gateway, and collaboration infrastructure;
- `research`: causal intelligence, evals, and research artifacts;
- `labs`: prototypes and seed demos;
- `docs`: public protocols, notes, and operating guides.

## 状态 / Status

早期公开 seed。WeHub 的公开组织会从小而有用的 artifact 开始生长，而不是一次性倾倒内部基础设施。

Early public seed. The organization will grow from small, useful artifacts rather than from a one-time dump of internal infrastructure.
